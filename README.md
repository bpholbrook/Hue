## Getting Started

### Installing

Clone the repository

```console
git clone https://github.com/bpholbrook/hue.git
```

Install the dependencies

```console
cd Hue
npm install
```

Launch the development server

```console
npm start
```

## Running the tests

### Test rendering

```console
npm test
```

### Test deployment

```console
npm run build
npm run serve
```

## Deployment

Deploy to GitHub Pages

```console
npm run deploy
```
